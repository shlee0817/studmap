\section{Collector}
\label{appendix:collector}
\subsection{Startbildschirm}
Nach Start der Anwendung befindet man sich auf dem Startbildschirm:
\begin{figure}[H]
\centering
\includegraphics[width=0.5\linewidth]{../Bilder/Collector/StartScreen}
\label{fig:StartScreen}
\end{figure}
Hier können Karte und Stockwerk für die Datenerfassung ausgewählt werden. Über den Button \textit{Laden} wird die entsprechende Auswahl geladen.

Hier sieht man das ausgewählte Stockwerk mit den zur Verfügung stehenden Punkten, die grün dargestellt sind.

\subsubsection{Punktauswahl}
Bei der Auswahl eines Punktes erscheint folgender Dialog:
\begin{figure}[H]
\centering
\includegraphics[width=0.35\linewidth]{../Bilder/Collector/StoreDataScreen}
\label{fig:StoreDataScreen}
\end{figure}
Über diesen Dialog können für den ausgewählten Punkt Positionsdaten hinterlegt werden. Dazu stehen die beiden Techniken NFC und WiFi zur Verfügung.

\paragraph{NFC-Tag Ansicht}
\textit{ }\\
Solange das Gerät keinen Kontakt zu einem NFC-Tag hat, wird nur die ID des Punktes ausgegeben.
\begin{figure}[H]
\centering
\includegraphics[width=0.35\linewidth]{../Bilder/Collector/NfcScreen}
\label{fig:NfcScreen}
\end{figure}

Sobald ein NFC-Tag gefunden wurde, wird auch die UID des NFC-Tags im Dialog angezeigt und der Datenpunkt kann mit dem NFC-Tag verknüpft werden.
\begin{figure}[H]
\centering
\includegraphics[width=0.35\linewidth]{../Bilder/Collector/NfcScreenRead}
\label{fig:NfcScreenRead}
\end{figure}
