\section{Client}
\begin{figure}[H]
\centering
\includegraphics[scale=0.5]{./Bilder/ClientGuide/ErsterStart.png}
\label{fig:ErsterStart}
\vspace{-10pt}
\end{figure}
Startet ein Benutzer die App erstmalig auf seinem Smartphone, so wird er von einem Willkommen-Dialog begrüßt. In diesem Dialog kann er die Karte auswählen, welche er nutzen möchte. Diese Auswahl kann später unter Einstellungen (siehe \ref{Einstellungen}) geändert werden. Zur Zeit steht für StudMap lediglich die Karte für die Westfälische Hochschule am Standort Bocholt zur Verfügung.

Nachdem die Auswahl erfolgt ist, wird die Karte geladen und die Applikation startet mit der Anzeige der untersten Ebene (hier Ebene 0).
\begin{figure}[H]
\centering
\includegraphics[scale=0.3]{./Bilder/ClientGuide/Start.png}
\caption{App nach erfolgreichem Start}
\label{fig:Startansicht}
\end{figure}

\label{ssec:StartZielpunkt}

Um einen Punkt als Start- oder Zielpunkt zu markieren, öffnet sich nach der Anwahl eines Punktes ein Dialog mit entsprechenden Auswahlmöglichkeiten.

\label{fig:StartZielWahl}

Wird ein Punkt durch die Suche oder den PoI-Dialog (siehe \nameref{PoIDialog}) markiert, so wird dieser leicht vergrößert und rot dargestellt. Im unteren Bildschirm wird zusätzlich ein Dialog mit den Knoteninformationen eingeblendet.

\subsubsection{Positionierung}
Die Positionierung kann auf zwei Arten erfolgen. Zum einen kann ein NFC-Tag eingescannt werden, dies erfolgt im Hintergrund und es ist keine Benutzeraktion notwendig. Zum anderen kann ein QR-Code eingescannt werden, wozu zunächst über das entsprechende Icon der Scanner gestartet werden muss.

\subsubsection{Neuladen}
Sollte die Karte einmal nicht erreichbar sein oder nur fehlerhaft geladen werden, so kann hier das Laden der Karte manuell angestoßen werden.
